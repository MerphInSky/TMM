# Blink Monitor

Приложение с использованием технологии компьютерного зрения для обнаружения глаз, определения и визуализации частоты моргания
глаз человека для определения факта засыпания по кадрам
видеопотока с видеокамеры в реальном масштабе времени с учетом
наличия и отсутствия прозрачных очков
